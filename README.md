### *We have five Brain Games:*

+ Brain even
+ Brain calc
+ Brain gcd
+ Brain progression
+ Brain prime

### *How to install games?*

In your terminal Print:
pip install -i https://test.pypi.org/simple/ ivan-ivan-t-brain-games --extra-index-url https://pypi.org/simple
